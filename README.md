# e-Devlet-Mobil-Uygulamas-Kullan-c-Yorumlar-n-n-NLP-ile-Duygu-Analizi-ve-Konu-Modellemesi
Google Play üzerinden elde edilen e-Devlet kullanıcı yorumlarının NLP teknikleri kullanılarak duygu analizi, konu modellemesi (LDA) ve makine öğrenmesi algoritmaları ile sınıflandırılmasını içeren proje.

Kullanılan Yöntemler
- Natural Language Processing (NLP)
- Text Preprocessing
- TF-IDF Vectorization
- N-gram (Unigram + Bigram)
- Sentiment Classification
- Logistic Regression, SVM, Naive Bayes
- Topic Modeling (LDA)


Süreç
 - Google Play verisi toplandı
 - Metin temizleme yapıldı
 - TF-IDF ile özellik çıkarımı yapıldı
 - Duygu sınıflandırma modelleri eğitildi
 - LDA ile konu analizi gerçekleştirildi

Sonuç
- En iyi model: Logistic Regression
- Accuracy: %86
- F1-Score: %85
